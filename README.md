# CPU Rasterizer

CPU-only 3D Rasterizer written in c++ without OpenGL, DirectX, or Vulcan. Implements the full CPU-side rendering pipeline: projection, triangle rasterization, z-buffering and shading.

Built to understand the math that GPU APIs abstract away.

##Why this exists

Educational deep dive into how rasterization works under the hood. GPU APIs hide this process in modern computing, 
I wanted to explore what they do exactly. I also wanted to produce authentic old-school rendering output using modern 
hardware for creative asset production. 

##Projected Features

- [ ] Frame buffer + PPM image output
- [ ] Triangle rasterization 
- [ ] Z-buffering / depth testing
- [ ] Perspective projection & camera
- [ ] .obj model loading
- [ ] Basic shading 
- [ ] Rotation / animation
- [ ] (stretch) Retro PS1-style mode: affine texture mapping, vertex 
      snapping, dithering
